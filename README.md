# Restaurant Roulette 🎰

A full-stack web application that helps users discover nearby restaurants at random.

Built with React and Express.

---

## Tech Stack

**Frontend**
- React

**Backend**
- Node.js
- Express

---

## Features

- Get nearby restaurants
- Apply filters
- Randomly select a restaurant

---

## Project Structure

restaurant-roulette/
├── client/
└── server/

---

## Running Locally

### Backend

cd server  
npm install  
npm run dev  

### Frontend

cd client  
npm install  
npm start
